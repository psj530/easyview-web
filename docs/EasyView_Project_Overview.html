1. 프로젝트 개요

PwC Easy View 3.0은 삼일회계법인(PwC Korea)의 Power BI 기반 재무분석 리포트를 웹 환경으로 구현한 플랫폼입니다. 기존 58페이지 PDF 리포트의 모든 분석 기능을 인터랙티브 웹 대시보드로 제공하며, 감사팀과 고객사가 실시간으로 재무 데이터를 분석하고 리포트를 생성할 수 있습니다.

핵심 목표

자동 리포트 생성
원본 분개장/시산표를 업로드하면 Easy View Template에 맞게 자동 가공하여 즉시 리포트 생성
접근 권한 통제
삼일회계법인 등록 계정만 접근 가능, 사용자별 담당 고객사 데이터만 노출
인터랙티브 분석
차트 클릭으로 Drill-down, 전표 검색, 시나리오 분석 등 실시간 상호작용
PDF 58페이지 대체
기존 정적 PDF 리포트를 동적 웹 대시보드로 완전 대체, 언제든 최신 데이터 반영

리포트 생성 프로세스

  1. 로그인 후 "새 리포트" 클릭
  2. 결산 연도/월 선택, 대상 회사 선택
  3. 분개장(JE/GL) + 시산표(TB) 필수 업로드, 재무제표/기타파일 선택 업로드
  4. "Report 생성하기" 클릭 → 리포트 변환 애니메이션 (단계별 진행)
  5. 생성 완료 → "Easy View 리포트 보기" → 대시보드에서 즉시 확인

5. 인증 & 보안

항목구현
비밀번호 해싱PBKDF2-HMAC-SHA256 (100,000 iterations + random salt)
토큰 방식JWT (HMAC-SHA256 서명, 24시간 만료)
접근 통제사용자별 회사 접근 권한 (user_companies 테이블)
자동 로그인선택적 자동 로그인 (localStorage)
보안 안내삼일회계법인 보안 정책 안내 메시지 표시
